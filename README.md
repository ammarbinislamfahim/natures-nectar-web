# Nature's Nectar Web

A modern invoice and inventory management system built with React and TypeScript.

## Features

- Dashboard with key metrics and charts
- Customer management
- Product inventory
- Invoice generation and management
- Reports and analytics

## Tech Stack

- React 18
- TypeScript
- Vite
- TailwindCSS
- shadcn/ui
- Dexie.js (IndexedDB)
- React Router DOM
- React Hook Form
- Zod
- Recharts
